# Step 1 — Identify & Qualify Dental Practices

> Before you reach out, know exactly who you're looking for. A targeted list beats a massive spray-and-pray list every time.

---

## 🎯 Ideal Prospect Profile (IPP)

Use this checklist to qualify a practice before adding it to your outreach list.

### Primary Qualifiers
- [ ] **Practice type:** General dentistry, orthodontics, cosmetic, or DSO
- [ ] **Practice size:** 1–5 dentists (single location) or 5+ (group/DSO)
- [ ] **Geography:** Define your target market (city, region, or national)
- [ ] **Revenue indicator:** Minimum $500K+ annual revenue (estimated via staff size and reviews)
- [ ] **Growth intent:** Recent expansion, new location, hiring activity, or new services advertised

### Secondary Qualifiers (Prioritization)
- [ ] Active Google Business Profile with recent reviews
- [ ] Running paid ads (Google/Facebook) — signals marketing spend and growth mindset
- [ ] Website that looks outdated or lacks clear patient conversion elements
- [ ] Low star rating (3–4 stars) with unanswered reviews — a clear gap you can help with
- [ ] Active on social media but inconsistent posting

- Search: `"dental practice" + [City]` or `"dentist" + "owner"`
- Filter by: 1st and 2nd connections, Company size: 2–50

## 📋 Prospect Scoring System

Score each lead 1–3 for each category. Prioritize prospects scoring **12+**.

| Criteria | 1 (Low) | 2 (Medium) | 3 (High) |
|----------|---------|------------|---------|
| Practice size | Solo dentist | 2–4 dentists | 5+ or DSO |
| Online reviews | <10 reviews | 10–50 reviews | 50+ reviews |
| Website quality | Professional | Average | Outdated/missing |
| Social media | Inactive | Occasional | Active |
| Hiring activity | None | 1 open role | Multiple roles |
| Growth signals | None | 1 signal | 2+ signals |

## 📁 Building Your Prospect List

### Recommended List Format

```
| Column          | Description                              |
|-----------------|------------------------------------------|
| Practice Name   | Official business name                   |
| Contact Name    | Decision-maker (owner, practice manager) |
| Title           | Dr., Practice Manager, CEO               |
| Email           | Verified direct email                    |
| Phone           | Main line + direct if available          |
| Location        | City, State                              |
| Website         | URL                                      |
| Google Rating   | 1–5 stars                                |
| Review Count    | Number of Google reviews                 |
| IPP Score       | Your qualification score (out of 18)     |
| Research Notes  | Key personalization notes                |
| Status          | New / Contacted / In Progress / Closed   |
```

### Tools to Manage Your List
- **Google Sheets** — free, easy to share
- **Airtable** — more structured, filterable views
- **HubSpot CRM (free tier)** — built-in email tracking
- **Notion** — great if you want docs + data together
