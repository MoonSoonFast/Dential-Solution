# 90-Day Client Success Plan

> The first 90 days define the relationship. This plan gives clients (and your team) a clear roadmap from onboarding to compounding results.

---

## Overview

| Phase | Days | Focus | Goal |
|-------|------|-------|------|
| Phase 1: Foundation | Days 1–30 | Fix the basics, establish systems | First visible results |
| Phase 2: Growth | Days 31–60 | Activate growth channels | Measurable new patient increase |
| Phase 3: Compounding | Days 61–90 | Optimize, scale, document wins | Sustainable momentum |

---

## Phase 1: Foundation (Days 1–30)

### Google Business Profile
- [ ] All information accurate and complete
- [ ] 25+ photos added (interior, exterior, team, procedure rooms)
- [ ] Services list fully populated with descriptions
- [ ] Business description rewritten with local keywords
- [ ] All existing unanswered reviews responded to
- [ ] Review request automation live and sending

### Website
- [ ] Baseline PageSpeed score documented
- [ ] Top 3 speed issues resolved
- [ ] "Book Online" CTA added above the fold (homepage + header)
- [ ] Online booking link active and tested
- [ ] Patient testimonials added to homepage
- [ ] Staff page with photos and bios live (if not present)

### Social Media
- [ ] Facebook and Instagram profiles optimized (bio, contact info, link in bio)
- [ ] 30-day content calendar built and approved
- [ ] First 2 weeks of content posted
- [ ] Comment/DM monitoring active

|--------|----------|---------------|
| Google star rating | | |
| Google review count | | |
| New patients/month | | |
| Website monthly visitors | | |
| Instagram followers | | |

---

## Phase 2: Growth (Days 31–60)

### Reputation
- [ ] 15+ new Google reviews collected since start
- [ ] Responding to all new reviews within 48 hours
- [ ] Healthgrades / Zocdoc profiles updated

### Patient Acquisition
- [ ] Google Ads campaign live (if in scope)
- [ ] Local SEO: top 3 directory listings claimed and verified
- [ ] Referral program communicated to front desk team
- [ ] Missed call follow-up process established

### Social Media
- [ ] Content cadence consistent (3+ posts/week)
- [ ] At least 1 patient testimonial post published (with consent)
- [ ] Instagram engagement growing month-over-month

### Reporting
- [ ] First monthly report delivered to client
- [ ] 30-day check-in call completed
- [ ] Updated priorities set for Days 31–60

---

## Phase 3: Compounding (Days 61–90)

### Optimization
- [ ] Top-performing content identified and doubled down on
- [ ] Google Ads CTR and conversion data reviewed (if applicable)
- [ ] Website analytics reviewed: top landing pages, bounce rates, booking funnel
- [ ] Review count growing consistently month-over-month

### Results Documentation
- [ ] 90-day results report prepared
- [ ] Before/after comparison documented
- [ ] Client testimonial or case study requested

### Renewal / Expansion
- [ ] 90-day performance review call scheduled
- [ ] Renewal conversation prepared
- [ ] Additional services presented if results warrant expansion

## 90-Day Success Metrics

| Metric | Typical Result |
|--------|---------------|
| New Google reviews | +25–50 |
| Google star rating change | +0.2–0.8 stars |
| New patients/month | +20–50% increase |
| Website bookings | +25–40% increase |
| Social media reach | +50–100% growth |

*(Results vary based on market, budget, and starting point)*

---

## Monthly Check-In Meeting Agenda

**Duration:** 30 minutes  
**Frequency:** Monthly (Day 14, 30, 60, 90)

1. Results review — what happened last month (10 min)
2. What we're doing next month and why (10 min)
3. Client Q&A and feedback (5 min)
4. Confirm any upcoming needs (holidays, promotions, events) (5 min)

---

## Sample Monthly Report Sections

1. **Executive Summary** — 3 bullets: top win, key metric, focus for next month
2. **Google Performance** — rating, reviews, profile views, calls
3. **Website Traffic** — visitors, bookings, conversion rate
4. **Social Media** — reach, engagement, follower growth
5. **New Patient Sources** — where new patients came from this month
6. **What's Next** — priorities for the coming month
