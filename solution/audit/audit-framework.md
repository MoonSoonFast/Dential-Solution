# Practice Audit Framework

> Use this framework to conduct a thorough assessment of any dental practice's online presence. The audit is the foundation for your free value offer AND your client proposals.

---

## When to Use This Audit

| Context | Purpose |
|---------|---------|
| **Pre-outreach research** | Identify personalization hooks before first contact |
| **Free audit offer** | Deliver live or via Loom to a prospect as a free value asset |
| **Proposal foundation** | Turn audit findings into a prioritized proposal |
| **Client onboarding** | Establish baseline metrics at the start of an engagement |
| **Monthly reporting** | Track progress against the baseline |

---

## Audit Categories & Scoring

Score each item 0–3:
- **3** = Excellent / fully in place
- **2** = Good with room to improve
- **1** = Poor / partially in place
- **0** = Missing entirely
